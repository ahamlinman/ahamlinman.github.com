    About

    I currently work on various hobby projects in my spare time, including...

    • CyTI
      • A Cython extension module to make the TiLP libraries available in Python and simpler to use. I've had something like this in the works for a while now, and I've tried a number of different plays on the concept. It has a bit of basic functionality and is currently a work in progress.
    • The Fruit Sale Service
      • A Google App Engine site for tracking a fundraiser. It's not currently in active development and is not designed for use by the general public. I may release it into the public domain at some point, since I completely rewrote the site for the December 2012 sale and might just give up on App Engine and switch to Rails in the future (if necessary).
    • Color of the Day
      • A small novelty website and app written for friends that provides a different color every day. The current web implementation is in PHP, though I have developed a new algorithm based on MD5 with a reference implementation in Java.

    I am hoping to start some more serious work in the future. My e-mail address is available on my GitHub profile.
